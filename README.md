# Description
C# toolkit for the judge0 REST API (https://api.judge0.com/).
The api is for running self-contained programs from various programming
languages (Java, Python, C#,...)

# Setup

Install .NET SDK 2.2

Build all projects:
```
$ dotnet build
```

Run test projects:
```
$ dotnet test
```

Run demo:
```
$ dotnet run --project Demo
```
